チュートリアル　プロジェクトの作成

プロジェクトフォルダの作成

プロジェクトの名前でフォルダを作成し、"template"フォルダの中身を丸ごとコピーしてください。
この"template"フォルダは、プロジェクトのひな形として用意しました。

メイクファイルの編集

`makefile'の先頭にある以下の定義を編集してください。
多くの場合、再設定が必要になります。

TOOL_PASS
各種ツール(*)があるパスを指定します。既にパスが通っている場合は、記述の必要はありません。
INC_PASS
本システムの`inc'フォルダのパスを指定してください。
LIB_PASS
本システムの`lib'フォルダのパスを指定してください。

設定例

TOOL_PASS = 
INC_PASS =  ..\inc\\
LIB_PASS =  ..\lib\\

TOOL_PASSについて

本'makefile'では、以下のツールを使います。
`TOOL_PASS'は、これらのツールがあるフォルダにパスが通っていない場合に、設定してください。

ca65.exe アセンブラ　（6502クロスコンパイラ`cc65'）
（アセンブリ言語のソースファイル(*.s)を、オブジェクトファイル(*.o)にアセンブリします。）
ar65.exe ライブラリアン　（6502クロスコンパイラ`cc65'）
（複数のオブジェクトファイル(*.o)を、一つのライブラリファイル(*.lib)にまとめます。）
ld65.exe リンカー　（6502クロスコンパイラ`cc65'）
（オブジェクトファイル(*.o)、ライブラリファイル(*.lib)から、ＲＯＭイメージファイル（*.nes）にビルドします。）
nsc.exe ＭＭＬコンパイラ　（音源ドライバ`nsd.lib'）
（ＭＭＬファイル（*.mml）から、アセンブリ言語のソースファイル(*.s)にコンパイルします。）
